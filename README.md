## 珠峰课堂接口

为珠峰课堂移动端提供后端接口

## 开发环境

- Node.js: `7.1.0`
- MongoDB: `2.6.7`
- Koa: `2.0.0`


## 如何使用
### 下载并启动
```
git clone https://github.com/zhufengketang/api.git
cd api
npm install
npm start
```
### 访问
```
http://localhost:3000/course
```
